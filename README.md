# Sujin Ryu — Portfolio

Personal portfolio site built with Next.js, showcasing UX engineering work across internal tools, consumer platforms, and AI interfaces.

🔗 [sujinryu.dev](https://sujinryu.dev)

## Tech Stack

- **Framework:** Next.js (App Router) + TypeScript
- **Styling:** SCSS Modules
- **Deployment:** Vercel
- **Domain:** Cloudflare

## Structure

src/
├── app/          Routes (/, /about, /work, /work/[slug], /contact)
├── components/   Cover, Nav, Footer, ProjectCard
├── data/         Project content (case studies)
└── styles/       Global SCSS variables

## Features

- Dynamic case study pages generated from a single data source
- Responsive layout across all breakpoints
- Keyboard-navigable interface

## Run Locally

npm install
npm run dev
